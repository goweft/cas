## Features Built

### Code Execution (`internal/runner/`)
- **Commit:** `734e564` (695 lines added, 43 new tests)
- Language detection from content: bash, Python, Go, JavaScript, Ruby
- Sandboxed subprocess with process group killing (`cmd.SysProcAttr.Setpgid`)
- 30-second timeout kills entire process tree via `syscall.Kill(-pid, SIGKILL)`
- Environment isolation: only PATH inherited, secrets don't leak
- Output formatting with truncation (4KB stdout, 2KB stderr)
- Intent detection: `KindRun` with anchored patterns ("run it", "execute", "test this")
- Shell wiring: validates workspace type (code only), content non-empty

### Lua Plugin Runtime (`internal/plugin/`)
- **Commit:** `56e9d2d` (863 lines added, 23 new tests)
- gopher-lua VM (pure Go Lua 5.1)
- Plugin discovery from `~/.cas/plugins/*.lua`
- Sandboxed: no `os`, no `io`, no `dofile`, no `loadfile`. Only base/table/string/math
- API: `cas.command(name, desc, fn)`, `cas.reply(text)`, `cas.workspaces()`, `cas.active()`
- Exact and prefix matching (case-insensitive)
- Error isolation: bad plugins don't block good ones
- Shell integration: plugins checked before intent detection (highest priority)
- `NewShellWithPlugins` constructor for custom plugin dirs in tests

### Cross-Workspace Operations (`internal/shell/resolve.go`)
- **Commit:** `57397c1` (600 lines added, 29 new tests)
- `resolveTarget()`: fuzzy-match workspace titles from messages (replaces hardcoded most-recent)
- `resolveAll()`: find all referenced workspaces for combine operations
- `crossWorkspaceRefs()`: find referenced workspaces for edit context injection
- Title scoring: exact match (100+len), partial word match (proportional), case-insensitive
- "all workspaces" / "everything" keyword detection
- New `KindCombine` intent: combine/merge/consolidate/put together
- `handleCombine`: resolves sources, builds multi-source LLM prompt, creates new document workspace
- Enhanced `handleEdit`: targets resolved workspace, includes cross-referenced content in LLM prompt
- New LLM functions: `BuildCombineMessages`, `BuildEditWithContextMessages`, `CombineSystem` prompt
